# Semantic Shores Vector Demo

Interactive demo of SQL Server 2025 vector search using semantic real estate search.

## Requirements

- SQL Server 2025 with vector search support
- Python 3.8+
- Flask, pyodbc, flask-cors

## Database

Requires the SemanticShoresDB sample database:
https://github.com/MrJoeSack/sqlserver-sample-databases/tree/master/semantic-shores-db

## Setup

```bash
pip install -r requirements.txt
python app.py
```

Open http://localhost:3000
